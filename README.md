# My First Data Science Project
# 💳 Credit Card Fraud Detection Pipeline

An end-to-end Machine Learning project built in Python to identify fraudulent credit card transactions using classification algorithms. This project focuses on handling highly imbalanced data, analyzing transaction patterns, and evaluating model performance using robust metrics.

## 🛠️ Tech Stack & Libraries
- **Language:** Python
- **Environment:** Jupyter Notebook
- **Data Wrangling:** Pandas, NumPy
- **Data Visualization:** Matplotlib, Seaborn
- **Machine Learning:** Scikit-Learn

## 🚀 Key Project Steps
- **Data Preprocessing:** Handled missing values, scaled transaction amounts, and addressed heavy class imbalance.
- **Exploratory Data Analysis (EDA):** Visualized transaction distributions, fraudulent anomalies, and correlations between features.
- **Model Training:** Trained and cross-validated multiple classification models (e.g., Logistic Regression, Random Forest).
- **Evaluation:** Evaluated models using Precision, Recall, F1-Score, and ROC-AUC curves rather than just basic accuracy due to the imbalanced nature of fraud datasets.

## 📊 Dataset Note
Due to GitHub's file size limitations, the raw data file `creditcard_2023.csv` (309 MB) is kept local and excluded from this repository.
